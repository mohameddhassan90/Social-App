# 🚀 Social App

A modern social media web application built using React and REST APIs.  
Users can create posts, follow other users, and interact in a dynamic feed.

## 🛠 Tech Stack

- React.js
- Vite
- React Query
- Axios
- Context API
- Tailwind CSS
- REST API

---

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 📝 Create, Edit & Delete Posts
- 👥 Follow / Unfollow Users
- 👤 User Profile Page
- 📱 Fully Responsive Design
- ⚡ Optimized Data Fetching with React Query

## 🖥 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/mohameddhassan90/Social-App.git
```

Go to the project folder:

```bash
cd Social-App
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
